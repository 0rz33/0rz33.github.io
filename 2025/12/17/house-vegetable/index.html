阳台生态循环系统完整方案

🌱 阳台生态循环系统完整方案（低成本 · 低维护 · 无异味）

目标：构建家庭阳台微型生态闭环

  • 厨余 → 肥料
  • 鱼缸 → 植物营养水
  • 水培 + 土培共存
  • 长期稳定、可持续

整体生态结构

阳台分区规划

🅐 厨余处理区（源头）

  • 密封式堆肥桶（防虫防臭）
  • 发酵液肥瓶（带放气）

🅑 土培区（主产出）

  • 花盆 / 种植箱
  • 使用自制堆肥 + 蛋壳粉

🅒 水培区（吸收营养水）

  • 静态水培瓶 / PVC 水培槽
  • 直接使用鱼缸换水

🅓 鱼缸区（生态核心）

  • 鱼缸 + 过滤 + 水草
  • 稳定供给氮源

厨余产生利用

分类原则

  • 能变肥的一定变肥
  • 需处理的先处理再进生态
  • 高风险物质分流

① 可直接堆肥（阳台生态能源）

植物性湿垃圾

  • 水果皮 / 果渣
  • 蔬菜边角 / 烂叶
  • 米饭、面食（少量）
  • 茶渣、咖啡渣

❌ 避免：

  • 高油、高盐、重辣

👉 处理方式：

  • 密封堆肥桶
  • 发酵液肥（1:20 稀释浇灌）

② 可加工再利用（处理后进生态）

🥚 鸡蛋壳

  • 清洗 → 晾干 → 炒干 → 碾粉
  • 作用：补钙、调节酸碱、防虫

🦐 虾壳 / 蟹壳（进阶）

  • 煮去盐 → 晒干 → 粉碎
  • 少量混土（缓释钙、甲壳素）

③ 可再利用但不进生态

🛍 塑料袋 / 包装袋

  • 分类收纳
  • 用途：垃圾袋、防潮垫、临时种植袋

④ 必须丢弃（不进阳台系统）

  • 重油肉汤
  • 高盐腌制品
  • 化学污染食物

水培营养液购买指南

基础步骤

  • （1）水培槽准备
    • 材质选择：塑料、亚克力、不锈钢均可
    • 深度：根系能伸展即可，一般20–30cm
    • 灌水/排水口设计，便于更换营养液
  • （2）水培基质
    • 常用材料：岩棉、椰砖、珍珠岩
    • 功能：固定植物，保持根部湿润
  • （3）营养液配比
    • 初学者可以购买成品水培营养液
    • 控制EC（电导率）和pH值：
      • pH：5.5–6.5
      • EC：1.0–2.0（根据植物不同调整）
  • （4）光照系统
    • LED生长灯（红蓝光）
    • 光照周期：
      • 叶菜类：12–16小时/天
      • 可用智能插座定时控制

中级水培种植

✅ 特点

  • 小型循环系统，提升效率
  • 半自动化管理
  • 可持续多种蔬菜

🍀 推荐蔬菜

  • 菠菜
  • 茼蒿
  • 芥蓝
  • 西洋菜
  • 紫苏 / 九层塔

🛠 方法

  • 系统：NFT（营养液膜技术）管道循环系统
  • 设备：微型水泵、定时器、LED 补光灯、pH/EC 计
  • 营养液管理：每 1014 天更换一次，EC 控制在 0.81.4，pH 在 5.5~6.5
  • 光照：自然光 + LED 补光（每天 8~12 小时）
  • 环境控制：建议放在阳台/封闭阳光房中

Arduino 自制 - 自动检测并起补充营养

传感器监控 + 泵时间控制 + EC/pH目标设定

🌿 系统结构（推荐架构）

模块 功能 建议型号
Arduino UNO / MEGA 主控制板 ESP32（集成Wi-Fi和蓝牙）
EC 传感器 检测营养液浓度 Gravity EC Sensor（DFRobot）
pH 传感器 检测酸碱度 Gravity pH Sensor
温度传感器 修正EC读数 DS18B20
蠕动泵 × 4 A液、B液、pH-、pH+ 12V 微型蠕动泵
继电器模块 控制泵通断 4通道继电器
搅拌马达（可选） 混合营养液 5V/12V 直流电机
LCD/OLED 屏幕 显示 EC、pH、状态 1602 或 0.96 OLED
按钮或Wi-Fi模块（ESP8266） 设定目标参数 方便远程调节

🥦 高级水培种植

✅ 特点

  • 全自动控制系统
  • 稳定高产，适合商业
  • 技术要求高，可远程管理

🍀 推荐蔬菜 / 作物

  • 番茄

  • 青椒、彩椒

🛠 方法

  • 系统类型：
    • DWC（深水培）
    • 滴灌式
    • 雾培（AeroPonics）
  • 控制系统：
    • 自动营养液配比
    • 温湿度传感器、pH/EC 自动监控
    • 接入 Home Assistant / Node-RED 实现自动化
  • 光照管理：LED植物灯 + 光照时控系统
  • CO₂/环境控制：封闭温室或小型 Grow Tent
