Ollama

2025-05-29 AI

Ollama 是一个基于 Go 语言开发的简单易用的本地大语言模型运行框架。
适用于macOS，Linux，Window，也可使用Docker

模型管理 ：

  • ollama list 显示模型列表。
  • ollama show 显示模型的信息
  • ollama pull 拉取模型
  • ollama push 推送模型
  • ollama cp 拷贝一个模型
  • ollama rm 删除一个模型
  • ollama run 运行一个模型
