技术指标 币种特性

核心区指标不变

  1. 核心趋势指标
    • EMA组合(10/50/100周期)
    • MACD(12/26/9参数) 金叉死叉
    • ADX趋势强度(14周期)
    • 多均线交叉信号(ema_cross)
  2. 动量震荡指标
    • RSI(14周期)
    • 随机指标(14,3,3)
    • CCI(14周期)
    • 价格变化率(return)
  3. 波动率管理指标
    • 布林带(20周期，2σ)
    • ATR真实波幅(14周期)
    • 布林带宽度特征(bb_width)
    • 价格波动率(14周期标准差)
  4. 成交量特征
    • 成交量Z分数(volume_zscore)
    • 成交量均值(14周期×1.5乘数)
    • VWAP(20周期)
  5. 特殊优化指标

持仓管理

flowchart TD
    A[获取当前账户状态\n（余额、持仓、价格、手续费）] --> B[判断市场信号\n（是否开仓、加仓、平仓）]

    B -->|开仓信号| C[是否已有持仓？]
    B -->|无信号或持仓中| Z[等待下一信号]

    C -->|已有持仓| D[判断是否满足加仓条件]
    C -->|无持仓| E[计算开仓资金与风险\n（资金占比、最大亏损等）]

    D --> F[计算加仓资金与风险\n（资金投入、风险敞口）]
    E --> G[执行开仓下单]

    F --> H[执行加仓下单]
    G --> I[更新持仓记录]
    H --> I

    I --> J[止盈 / 止损 / 换仓判断]
    J --> K[是否满足平仓条件？]
    K -->|是| L[计算当前盈亏与风险比\n（盈亏率、RR比、最大回撤）]
    K -->|否| Z

    L --> M[执行平仓 / 止损 / 减仓下单]
    M --> N[更新资金与交易日志]

xgboost心得

  1. 注意币种特性
  2. 用xgboots分析市场入场信号
    • 标签可以更细化
    • 入场信号细化
    • 权重必不可少 因为交易都有冲突
  3. 入场后，交易中，离场
    • 动态止顺止盈，持仓情况
    • 风控要做好
    • 权重必不可少 因为交易都有冲突
    • 模型可以随时调整
    • 多指标融合 不同策略
  4. xgboost训练数据
    • 训练加入监控
    • 调整相应参数
