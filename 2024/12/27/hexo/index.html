  • 创建个人访问令牌（Personal Access Token） – blog项目

1 点击右上角头像 -> 打开 Settings -> 左边栏滚到最后找到 Develop Setting 打开，如图

2 找到 Personal Access Token 点击 Tokens（classic） -> 选择 Generate new token (classic) ，如图

3 给令牌起个名字并且勾选红框中的 repo 的访问权限，然后点击生成 token（注意： 确保复制并安全地保存这个令牌。GitHub 不会再次显示这个令牌，所以这是你唯一的机会复制它），如图

4 将生成的 PAT 添加到你的博客源代码仓库的 Secrets，名字填入 PERSONAL_TOKEN 后面会用到这个变量名，如图
